This tool is based on Mythic Game Master Emulator, Mythic Variations, Mythic Variations 2, and the Mythic Magazines by Tana Pigeon. Visit www.wordmillgames.com for more information and lots of awesomeness.

Note you can resize the "Inspiration goes here" box, write your notes in there and, when you are finished, copy them to your note taking app of choice.

This is just a single html file, so you can use it on any device with a (modern) web browser, even offline. Just right-click anywhere, then select "Save as", then enter a cool file name and select "Web Page, complete (*.htm;*.html)" and copy the file to the device where you want to use it.
